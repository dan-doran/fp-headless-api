In this directory are the Wordpress / Woocommerce plugins that are behind a paywall or otherwise not directly available to install. The **best practice** for upgrading these plugins is to
1. Log into the vendor website
2. Download the plugin
3. ZIP if needed
4. Replace the ZIP file in this directory
5. Redeploy
